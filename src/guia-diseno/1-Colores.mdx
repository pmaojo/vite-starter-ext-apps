# Sistema de Colores

Nuestra paleta de colores es la base visual de la aplicación. Está diseñada para proporcionar un contraste adecuado (cumpliendo con estándares de accesibilidad) y comunicar intención y jerarquía.

## Consejos para Desarrolladores

Para utilizar estos colores en componentes de Tailwind CSS, simplemente usa las clases de utilidad generadas. Por ejemplo:

- `bg-primary` para aplicar el color de fondo primario.
- `text-destructive` para aplicar el color de texto rojo de advertencia.
- `border-border` para aplicar el color sutil de los bordes.

Todos estos colores reaccionan automáticamente al tema (Claro / Oscuro) definido por variables CSS en `src/global.css`.
