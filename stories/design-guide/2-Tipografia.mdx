# Sistema Tipográfico

La tipografía es fundamental para la legibilidad y la jerarquía de la información. Nuestro proyecto utiliza fuentes modernas, optimizadas para pantallas digitales (como Geist o Inter).

**Familia Tipográfica Principal:** Sans-serif (Geist Variable)

## Jerarquía Visual

Utilizamos clases de utilidad de Tailwind CSS para aplicar tamaños, pesos y alturas de línea (line-height). A continuación, se muestran los tamaños de texto estándar utilizados en la interfaz.

## Notas para el Diseño

Al diseñar nuevas vistas, asegúrate de no saltar niveles jerárquicos (ej. de H1 a H3 sin pasar por H2). El contraste en el tamaño de la fuente es clave para guiar el ojo del usuario a través del contenido de manera intuitiva.
